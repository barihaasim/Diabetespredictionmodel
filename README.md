Diabetes Prediction Project
Overview
This project uses machine learning to predict if a patient has diabetes based on health data like glucose levels, BMI, and age. It includes a trained model, the dataset, and a web application script.

Project Contents
app.py: The script to run the web application.

main.ipynb: The notebook where the data was analyzed and the model was built.

diabetes_model.pkl: The saved machine learning model.

scaler.pkl: The tool used to process the data before prediction.

diabetes.csv: The patient data used for training.

The ui is made on streamlit..
